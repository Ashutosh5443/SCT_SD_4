# E-Commerce Product Scraper 
Overview

This Python program extracts product information (such as names, prices, and ratings) from an online e-commerce website and stores the data in a structured format like a CSV file. It uses web scraping techniques to collect real-time product data, making it useful for price comparisons, market analysis, and tracking product trends.

Features

✅ Extracts product details (name, price, rating, and more)

✅ Saves data to CSV for easy analysis

✅ Handles pagination to scrape multiple pages

✅ User-agent rotation to prevent detection

✅ Error handling for failed requests
